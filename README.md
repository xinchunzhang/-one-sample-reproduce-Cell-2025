# -one-sample-reproduce-Cell-2025
Reproduce analysis of [An Arabidopsis single-nucleus atlas decodes leaf senescence and nutrient allocation] single-cell paper using one sample. 
**Authors:** Guo, X., Wang, Y., Zhao, C., ..., Liu, H., Guo, H.W., Xu, X.  
**DOI:** https://doi.org/10.1016/j.cell.2025.03.024

## Project Goal
This repository contains the code to reproduce the single-cell/nucleus analysis from the above paper, specifically focusing on decoding leaf senescence and nutrient allocation in Arabidopsis using a single sample.

## Key Features of the Original Study
- Constructed a single-nucleus transcriptomic atlas across 20 tissues (>1 million nuclei).
- Formulated SAG-index and YAG-index to quantify leaf aging at single-cell level.
- Mapped nutrient allocation (source-to-sink) between leaves and other organs.
